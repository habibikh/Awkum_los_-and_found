AWKUM Lost and Found Application

Overview

The AWKUM Lost and Found Application is a web-based system designed to help students, faculty, and staff of Abdul Wali Khan University Mardan (AWKUM) report, search, and recover lost items efficiently.
The application leverages Streamlit for the user interface, Google Colab for development and deployment, and Groq LLM for intelligent text processing and matching.


Features

Report lost items with detailed descriptions

Report found items

Intelligent matching using AI (Groq)

Simple and user-friendly Streamlit interface

Fast deployment and testing using Google Colab

Centralized record of lost and found items
